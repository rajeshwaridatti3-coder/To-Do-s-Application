# ✅ To-Do Application

## 📖 Overview
A responsive To-Do Application developed using HTML, CSS, and JavaScript that helps users organize and manage daily tasks efficiently. The application allows users to add, complete, and delete tasks while maintaining task data using Local Storage.

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persistent task storage using Local Storage
- Responsive and user-friendly interface
- Simple task management system

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Local Storage API

## 🎯 Functionality
- Create and manage daily tasks
- Track completed and pending tasks
- Store task data locally in the browser
- Maintain tasks even after page refresh
